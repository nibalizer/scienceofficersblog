GPG stands for Gnu Privacy Guard, PGP stands for Pretty Good privacy. These two tools are interchangeable.
My long fingerprint is: 7857 DAF4 33DE D013 A6CE B5DB 4A33 0B9A D648 FAE3

I have several emails attached to my key. All work but the best way to reach me is using nibz@spencerkrum.com This is the uid that I have set as my primary key, and the uid that I commit with in software projects.
Recently I used gpg to encrypt the credentials for the Puppet Forge for puppet-community.
This file https://github.com/puppet-community/plumbing/blob/master/puppet_credentials.asc can only be accessed by me and Igor Galic. Separately we can both decrypt the file, grab the credentials (note the helper scripts also in that directory) and publish puppet modules to the Puppet forge.
Recently I went to the FOSDEM gpg keysigning party. Over 200 people attended. Read more about it here: https://fosdem.org/2015/keysigning/
I have recieved more gpg key signatures from this one event than the rest of the signatures on my key combined.
I hope to use gpg in the future for signing deb/rpm packages, for signing git tags, and for securely communicating.
